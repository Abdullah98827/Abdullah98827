## Projects

**[Teacher-Hub](https://github.com/Abdullah98827/teacher-hub)** — Secure 
cross-platform application for UK teachers to share and collaborate on 
educational resources, with AI accessibility tools for SEN and EAL learners. 
Built in React Native and Supabase with JWT authentication, MFA, TRN-based 
identity verification, and row-level security. During security testing I 
identified and patched a real OWASP A02:2021 (Cryptographic Failures) 
vulnerability. Achieved 93/100 SUS score across 279 test cases. 
Dissertation grade: 86%.

**[VerifyDoc](https://github.com/Abdullah98827/verifydoc)** — Python CLI 
tool implementing Ed25519 digital signatures and SHA-256 hashing for document 
authenticity and batch tamper detection. Achieved 100% tamper detection across 
70-100 test documents. Applied STRIDE threat modelling and mapped to UK GDPR 
Article 32 and ISO 27001 Annex A.8.24.

**[CityFix](https://github.com/Abdullah98827/CityFix)** — Council issue 
reporting mobile app built in React Native and Expo with Firebase. Five user 
roles with role-based access control, real-time GPS, Haversine-based duplicate 
detection, and 100+ test cases covering authentication flows and data integrity.

**Penetration Testing Engagement (University Lab)** — Full internal network 
penetration test from pre-engagement planning through to structured remediation 
reporting. Scope covered five hosts across Windows and Linux environments.

Methodology: passive and active reconnaissance with Nmap, vulnerability 
scanning with Nessus, exploitation with Metasploit.

Key findings:
- Root shell access via vsFTPd 2.3.4 backdoor on Metasploitable
- Meterpreter session via EternalBlue (MS17-010) on Windows 7 with 
  post-exploitation enumeration
- CVE-2019-0708 (BlueKeep) identified on RDP service
- Apache PHP CGI remote code execution (CVSS 9.8)
- SQL injection in phpMyAdmin
- Directory traversal and clickjacking on Apache using Gobuster and Metasploit

All findings documented in a structured report with executive summary, 
CVSS-rated vulnerability tables, exploitation screenshots, and prioritised 
remediation recommendations.
